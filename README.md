**게임 요약**
카드를 뽑아서 패시브를 획득하고 보스와 전투를 벌이는 게임.

**1. 카드 시스템**
노말, 커먼, 스페셜 등급의 총 10장의 카드 중 랜덤으로 3장을 뽑아 패시브를 얻을 수 있다. 단, 스페셜 등급의 카드는 한 장만 가능하다.
